# Textpro

TextPro is a little command line python program that creates a dialogue via command line.

### Features:
-Detects questions if the phrase starts with an interrogative pronoun
-Capitalizes all strings inserted

TextPro creates a dialogue via user input.

A dialogue for example can be as:
### Input
```
Say something: hello there
Say somehting: how are you
Say something: i'm fine
Say something: thanks
Say something: \end
```
To end the program, the user must type:
```
\end
```
### Output
```
Hello There.
How are you?
I'm fine, thanks.
```
### Installation

To install TextPro, the user must clone the following repository:
https://github.com/MatMatias/TextPro

### Usage

To use Textpro, the user must execute it via terminal.

To do that, the user must open a terminal on the folder that was cloned. With terminal opened, execute TextPro by typing the following command:

Windows users:
```
 $ py textpro.py
```
Linux users:
```
$ python textpro.py
```
